## Content

VK_HEADER_VERSION - Vulkan header file version number

`VK_HEADER_VERSION` is the version number of the `vulkan_core.h` header.
This value is kept synchronized with the patch version of the released
Specification.

// Provided by VK_VERSION_1_0
// Version of this file
#define VK_HEADER_VERSION 335

[VK_HEADER_VERSION_COMPLETE](VK_HEADER_VERSION_COMPLETE.html), [VK_VERSION_1_0](VK_VERSION_1_0.html)

For more information, see the [Vulkan Specification](../../../../spec/latest/appendices/boilerplate.html#VK_HEADER_VERSION).
