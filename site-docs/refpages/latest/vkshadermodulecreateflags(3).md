VkShaderModuleCreateFlags - Reserved for future use

// Provided by VK_VERSION_1_0
typedef VkFlags VkShaderModuleCreateFlags;

`VkShaderModuleCreateFlags` is a bitmask type for setting a mask, but is
currently reserved for future use.

[VK_VERSION_1_0](VK_VERSION_1_0.html), `VkFlags`, [VkShaderModuleCreateInfo](VkShaderModuleCreateInfo.html)

For more information, see the [Vulkan Specification](../../../../spec/latest/chapters/shaders.html#VkShaderModuleCreateFlags).
