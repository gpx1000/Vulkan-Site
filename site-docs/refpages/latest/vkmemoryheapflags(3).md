VkMemoryHeapFlags - Bitmask of VkMemoryHeapFlagBits

// Provided by VK_VERSION_1_0
typedef VkFlags VkMemoryHeapFlags;

`VkMemoryHeapFlags` is a bitmask type for setting a mask of zero or more
[VkMemoryHeapFlagBits](VkMemoryHeapFlagBits.html).

[VK_VERSION_1_0](VK_VERSION_1_0.html), `VkFlags`, [VkMemoryHeap](VkMemoryHeap.html), [VkMemoryHeapFlagBits](VkMemoryHeapFlagBits.html)
