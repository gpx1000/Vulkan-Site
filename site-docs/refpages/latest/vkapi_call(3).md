VKAPI_CALL - Vulkan function calling conventions macro

`VKAPI_CALL` is a macro placed after the return type in Vulkan API
function declarations.
This macro controls calling conventions for MSVC-style compilers.

[VKAPI_ATTR](VKAPI_ATTR.html), [VKAPI_PTR](VKAPI_PTR.html)

For more information, see the [Vulkan Specification](../../../../spec/latest/appendices/boilerplate.html#boilerplate-platform-specific-calling-conventions).
