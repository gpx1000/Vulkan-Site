VkVideoEncodeUsageFlagsKHR - Bitmask specifying the video encode usage flags

// Provided by VK_KHR_video_encode_queue
typedef VkFlags VkVideoEncodeUsageFlagsKHR;

`VkVideoEncodeUsageFlagsKHR` is a bitmask type for setting a mask of
zero or more [VkVideoEncodeUsageFlagBitsKHR](VkVideoEncodeUsageFlagBitsKHR.html).

[VK_KHR_video_encode_queue](VK_KHR_video_encode_queue.html), `VkFlags`, [VkVideoEncodeUsageFlagBitsKHR](VkVideoEncodeUsageFlagBitsKHR.html), [VkVideoEncodeUsageInfoKHR](VkVideoEncodeUsageInfoKHR.html)
