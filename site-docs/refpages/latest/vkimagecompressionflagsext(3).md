VkImageCompressionFlagsEXT - Bitmask of VkImageCompressionFlagBitsEXT

// Provided by VK_EXT_image_compression_control
typedef VkFlags VkImageCompressionFlagsEXT;

`VkImageCompressionFlagsEXT` is a bitmask type for setting a mask of
zero or more [VkImageCompressionFlagBitsEXT](VkImageCompressionFlagBitsEXT.html).

[VK_EXT_image_compression_control](VK_EXT_image_compression_control.html), `VkFlags`, [VkImageCompressionControlEXT](VkImageCompressionControlEXT.html), [VkImageCompressionFlagBitsEXT](VkImageCompressionFlagBitsEXT.html), [VkImageCompressionPropertiesEXT](VkImageCompressionPropertiesEXT.html)
