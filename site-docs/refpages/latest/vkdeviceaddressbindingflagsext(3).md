VkDeviceAddressBindingFlagsEXT - Bitmask of VkDeviceAddressBindingFlagBitsEXT

// Provided by VK_EXT_device_address_binding_report
typedef VkFlags VkDeviceAddressBindingFlagsEXT;

[VkDeviceAddressBindingFlagsEXT](#) is a bitmask type for setting a mask of
zero or more [VkDeviceAddressBindingFlagBitsEXT](VkDeviceAddressBindingFlagBitsEXT.html).
