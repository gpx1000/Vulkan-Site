VkAccessFlags3KHR - 64-bit mask of access flags

`VkAccessFlags3KHR` is a bitmask type for setting a mask of zero or more
[VkAccessFlagBits3KHR](VkAccessFlagBits3KHR.html):

typedef VkFlags64 VkAccessFlags3KHR;
