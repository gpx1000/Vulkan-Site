VkConditionalRenderingFlagsEXT - Bitmask of VkConditionalRenderingFlagBitsEXT

// Provided by VK_EXT_conditional_rendering
typedef VkFlags VkConditionalRenderingFlagsEXT;

`VkConditionalRenderingFlagsEXT` is a bitmask type for setting a mask of
zero or more [VkConditionalRenderingFlagBitsEXT](VkConditionalRenderingFlagBitsEXT.html).

[VK_EXT_conditional_rendering](VK_EXT_conditional_rendering.html), [VkConditionalRenderingBeginInfoEXT](VkConditionalRenderingBeginInfoEXT.html), [VkConditionalRenderingFlagBitsEXT](VkConditionalRenderingFlagBitsEXT.html), `VkFlags`
