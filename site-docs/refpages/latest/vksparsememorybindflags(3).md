VkSparseMemoryBindFlags - Bitmask of VkSparseMemoryBindFlagBits

// Provided by VK_VERSION_1_0
typedef VkFlags VkSparseMemoryBindFlags;

`VkSparseMemoryBindFlags` is a bitmask type for setting a mask of zero
or more [VkSparseMemoryBindFlagBits](VkSparseMemoryBindFlagBits.html).

[VK_VERSION_1_0](VK_VERSION_1_0.html), `VkFlags`, [VkSparseImageMemoryBind](VkSparseImageMemoryBind.html), [VkSparseMemoryBind](VkSparseMemoryBind.html), [VkSparseMemoryBindFlagBits](VkSparseMemoryBindFlagBits.html)
