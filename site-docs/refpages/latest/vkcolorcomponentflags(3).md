VkColorComponentFlags - Bitmask of VkColorComponentFlagBits

// Provided by VK_VERSION_1_0
typedef VkFlags VkColorComponentFlags;

`VkColorComponentFlags` is a bitmask type for setting a mask of zero or
more [VkColorComponentFlagBits](VkColorComponentFlagBits.html).

[VK_VERSION_1_0](VK_VERSION_1_0.html), [VkColorComponentFlagBits](VkColorComponentFlagBits.html), `VkFlags`, [VkPipelineColorBlendAttachmentState](VkPipelineColorBlendAttachmentState.html), [vkCmdSetColorWriteMaskEXT](vkCmdSetColorWriteMaskEXT.html)
