VkSubgroupFeatureFlags - Bitmask of VkSubgroupFeatureFlagBits

// Provided by VK_VERSION_1_1
typedef VkFlags VkSubgroupFeatureFlags;

`VkSubgroupFeatureFlags` is a bitmask type for setting a mask of zero or
more [VkSubgroupFeatureFlagBits](VkSubgroupFeatureFlagBits.html).

[VK_VERSION_1_1](VK_VERSION_1_1.html), `VkFlags`, [VkPhysicalDeviceSubgroupProperties](VkPhysicalDeviceSubgroupProperties.html), [VkPhysicalDeviceVulkan11Properties](VkPhysicalDeviceVulkan11Properties.html), [VkSubgroupFeatureFlagBits](VkSubgroupFeatureFlagBits.html)
