VkCommandPoolResetFlags - Bitmask of VkCommandPoolResetFlagBits

// Provided by VK_VERSION_1_0
typedef VkFlags VkCommandPoolResetFlags;

`VkCommandPoolResetFlags` is a bitmask type for setting a mask of zero
or more [VkCommandPoolResetFlagBits](VkCommandPoolResetFlagBits.html).

[VK_VERSION_1_0](VK_VERSION_1_0.html), [VkCommandPoolResetFlagBits](VkCommandPoolResetFlagBits.html), `VkFlags`, [vkResetCommandPool](vkResetCommandPool.html)

For more information, see the [Vulkan Specification](../../../../spec/latest/chapters/cmdbuffers.html#VkCommandPoolResetFlags).
