VkPresentTimingInfoFlagsEXT - Bitmask of VkPresentTimingInfoFlagBitsEXT

// Provided by VK_EXT_present_timing
typedef VkFlags VkPresentTimingInfoFlagsEXT;

`VkPresentTimingInfoFlagsEXT` is a bitmask type for setting a mask of
zero or more [VkPresentTimingInfoFlagBitsEXT](VkPresentTimingInfoFlagBitsEXT.html).

[VK_EXT_present_timing](VK_EXT_present_timing.html), `VkFlags`, [VkPresentTimingInfoEXT](VkPresentTimingInfoEXT.html), [VkPresentTimingInfoFlagBitsEXT](VkPresentTimingInfoFlagBitsEXT.html)
