VkVideoEncodeAV1RateControlFlagsKHR - Bitmask specifying AV1 encode rate control flags

// Provided by VK_KHR_video_encode_av1
typedef VkFlags VkVideoEncodeAV1RateControlFlagsKHR;

`VkVideoEncodeAV1RateControlFlagsKHR` is a bitmask type for setting a
mask of zero or more [VkVideoEncodeAV1RateControlFlagBitsKHR](VkVideoEncodeAV1RateControlFlagBitsKHR.html).
