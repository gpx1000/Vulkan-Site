VkPastPresentationTimingFlagsEXT - Bitmask of VkPastPresentationTimingFlagBitsEXT

// Provided by VK_EXT_present_timing
typedef VkFlags VkPastPresentationTimingFlagsEXT;

`VkPastPresentationTimingFlagsEXT` is a bitmask type for setting a mask
of zero or more [VkPastPresentationTimingFlagBitsEXT](VkPastPresentationTimingFlagBitsEXT.html).

[VK_EXT_present_timing](VK_EXT_present_timing.html), `VkFlags`, [VkPastPresentationTimingFlagBitsEXT](VkPastPresentationTimingFlagBitsEXT.html), [VkPastPresentationTimingInfoEXT](VkPastPresentationTimingInfoEXT.html)
