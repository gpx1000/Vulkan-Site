VkVideoEncodeH264RateControlFlagsKHR - Bitmask specifying H.264 encode rate control flags

// Provided by VK_KHR_video_encode_h264
typedef VkFlags VkVideoEncodeH264RateControlFlagsKHR;

`VkVideoEncodeH264RateControlFlagsKHR` is a bitmask type for setting a
mask of zero or more [VkVideoEncodeH264RateControlFlagBitsKHR](VkVideoEncodeH264RateControlFlagBitsKHR.html).
