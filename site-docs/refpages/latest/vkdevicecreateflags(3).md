VkDeviceCreateFlags - Reserved for future use

// Provided by VK_VERSION_1_0
typedef VkFlags VkDeviceCreateFlags;

`VkDeviceCreateFlags` is a bitmask type for setting a mask, but is
currently reserved for future use.

[VK_VERSION_1_0](VK_VERSION_1_0.html), [VkDeviceCreateInfo](VkDeviceCreateInfo.html), `VkFlags`

For more information, see the [Vulkan Specification](../../../../spec/latest/chapters/devsandqueues.html#VkDeviceCreateFlags).
