VkDependencyFlags - Bitmask of VkDependencyFlagBits

// Provided by VK_VERSION_1_0
typedef VkFlags VkDependencyFlags;

`VkDependencyFlags` is a bitmask type for setting a mask of zero or more
[VkDependencyFlagBits](VkDependencyFlagBits.html).

[VK_VERSION_1_0](VK_VERSION_1_0.html), [VkDependencyFlagBits](VkDependencyFlagBits.html), [VkDependencyInfo](VkDependencyInfo.html), `VkFlags`, [VkSubpassDependency](VkSubpassDependency.html), [VkSubpassDependency2](VkSubpassDependency2.html), [vkCmdPipelineBarrier](vkCmdPipelineBarrier.html)
