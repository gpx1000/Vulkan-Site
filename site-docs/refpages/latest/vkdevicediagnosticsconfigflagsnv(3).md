VkDeviceDiagnosticsConfigFlagsNV - Bitmask of VkDeviceDiagnosticsConfigFlagBitsNV

// Provided by VK_NV_device_diagnostics_config
typedef VkFlags VkDeviceDiagnosticsConfigFlagsNV;

`VkDeviceDiagnosticsConfigFlagsNV` is a bitmask type for setting a mask
of zero or more [VkDeviceDiagnosticsConfigFlagBitsNV](VkDeviceDiagnosticsConfigFlagBitsNV.html).
