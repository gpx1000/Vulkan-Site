VkVideoEncodeH265RateControlFlagsKHR - Bitmask specifying H.265 encode rate control flags

// Provided by VK_KHR_video_encode_h265
typedef VkFlags VkVideoEncodeH265RateControlFlagsKHR;

`VkVideoEncodeH265RateControlFlagsKHR` is a bitmask type for setting a
mask of zero or more [VkVideoEncodeH265RateControlFlagBitsKHR](VkVideoEncodeH265RateControlFlagBitsKHR.html).
