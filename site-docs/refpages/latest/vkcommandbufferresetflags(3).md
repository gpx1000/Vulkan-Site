VkCommandBufferResetFlags - Bitmask of VkCommandBufferResetFlagBits

// Provided by VK_VERSION_1_0
typedef VkFlags VkCommandBufferResetFlags;

`VkCommandBufferResetFlags` is a bitmask type for setting a mask of zero
or more [VkCommandBufferResetFlagBits](VkCommandBufferResetFlagBits.html).

[VK_VERSION_1_0](VK_VERSION_1_0.html), [VkCommandBufferResetFlagBits](VkCommandBufferResetFlagBits.html), `VkFlags`, [vkResetCommandBuffer](vkResetCommandBuffer.html)

For more information, see the [Vulkan Specification](../../../../spec/latest/chapters/cmdbuffers.html#VkCommandBufferResetFlags).
