VkRenderingAttachmentFlagsKHR - Bitmask of VkRenderingAttachmentFlagBitsKHR

// Provided by VK_KHR_maintenance10
typedef VkFlags VkRenderingAttachmentFlagsKHR;

`VkRenderingAttachmentFlagsKHR` is a bitmask type for setting a mask of
zero or more [VkRenderingAttachmentFlagBitsKHR](VkRenderingAttachmentFlagBitsKHR.html).
