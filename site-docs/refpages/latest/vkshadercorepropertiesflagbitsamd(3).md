## Content

VkShaderCorePropertiesFlagBitsAMD - Bitmask specifying shader core properties

Bits for this type **may** be defined by future extensions, or new versions of
the `[VK_AMD_shader_core_properties2](VK_AMD_shader_core_properties2.html)` extension.
Possible values of the `flags` member of
[VkShaderCorePropertiesFlagsAMD](VkShaderCorePropertiesFlagsAMD.html) are:

// Provided by VK_AMD_shader_core_properties2
typedef enum VkShaderCorePropertiesFlagBitsAMD {
} VkShaderCorePropertiesFlagBitsAMD;

[VK_AMD_shader_core_properties2](VK_AMD_shader_core_properties2.html), [VkPhysicalDeviceShaderCoreProperties2AMD](VkPhysicalDeviceShaderCoreProperties2AMD.html), [VkShaderCorePropertiesFlagsAMD](VkShaderCorePropertiesFlagsAMD.html)

For more information, see the [Vulkan Specification](../../../../spec/latest/chapters/limits.html#VkShaderCorePropertiesFlagBitsAMD).
