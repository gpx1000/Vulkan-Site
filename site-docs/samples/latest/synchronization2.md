The source for this sample can be found in the Khronos Vulkan samples github repository. Extension VK_KHR_synchronization2 Demonstrates the use of the reworked synchronization api introduced with VK_KHR_synchronization2. Based on the compute shading N-Body particle system, this sample uses the new extension to streamline the memory barriers used for the compute and graphics work submissions.
