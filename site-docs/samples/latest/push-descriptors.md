## Content

The source for this sample can be found in the Khronos Vulkan samples github repository. Extension: VK_KHR_push_descriptor Push descriptors apply the push constants concept to descriptor sets. Instead of creating per-object descriptor sets, this example passes descriptors at command buffer creation time.
